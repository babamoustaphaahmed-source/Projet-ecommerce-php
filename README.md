## Installation

1. Copier dans `C:\xampp\htdocs\`

2. Lancer XAMPP Apache + MySQL  

3. Créer base `restaurant` sur phpMyAdmin

4. Importer `database/restaurant.sql`

5. Aller sur `localhost/restaurant/login.php`
